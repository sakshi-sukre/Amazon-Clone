# Amazon Clone

A responsive Amazon-inspired e-commerce website built using HTML and CSS. This project recreates the user interface of Amazon's homepage, including navigation menus, search functionality, product sections, promotional banners, and footer layouts.

## Features

* Responsive navigation bar
* Search bar with category selection
* Language selector
* Hero banner section
* Product showcase cards
* Product image galleries
* Hover effects and interactive UI elements
* Multi-section footer
* Amazon-inspired layout and styling
* Responsive design using Flexbox and CSS Grid

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Font Awesome
